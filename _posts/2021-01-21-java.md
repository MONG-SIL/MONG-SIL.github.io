자바 스크립트 언어는 동적이기 때문에 웹 페이지에 원하는 정보를 마음껏 담을 수 있다.

.을 이용해 함수 사용 가능
다른 프로그래밍 언어와 비슷한 느낌이다.

## Boolean
'==='는 좌항과 우항이 같은지 비교하는 비교연산자이다.
이 비교연산자를 사용하면 True or False 둘 중 한가지의 값이 나오는데 이 두 값을 Boolean 데이터 타입이라고 한다.
